# p5js-snooker

🎱 **World Snooker Championship** — browser game built with p5.js

## Play
`ash
npx serve .
`
Then open http://localhost:3000

## Features
- Full 22-ball official snooker setup
- Official scoring (red=1, yellow=2, green=3, brown=4, blue=5, pink=6, black=7)
- Official turn management with fouls and ball-in-hand
- Smart computer opponent (AI shot calculation)
- Mouse click & drag cue control
- Sound effects via Web Audio API
- Responsive canvas
- Dramatic WK Snooker startup screen (Crucible, Sheffield)

## Stack
- [p5.js](https://p5js.org/) — rendering & input
- Web Audio API — sound synthesis
- Vanilla JS modules
